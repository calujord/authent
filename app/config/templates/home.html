Capacidades

Todo lo que necesitas para autenticar

Construido sobre Django REST Framework con JWT y soporte multi-tenant.

🔐
JWT Seguro
Autenticación basada en tokens JWT con refresh automático y blacklist de tokens revocados.
👥
Gestión de usuarios
Registro, login, recuperación de contraseña, invitaciones y roles por aplicación.
🏢
Multi-tenant
Soporte para múltiples aplicaciones con API keys independientes y aislamiento de datos.
📧
Email integrado
Envío de correos para verificación, invitaciones y notificaciones vía Resend.
⚡
Tareas asíncronas
Celery + Redis para procesar tareas en segundo plano sin bloquear las respuestas.
📖
API documentada
Esquema OpenAPI 3.0 generado automáticamente con Swagger UI y ReDoc integrados.
